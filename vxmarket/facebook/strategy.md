## استراتيجية الإعلانات المدفوعة

### الحملة 1: Brand Awareness (توعية)
```
الهدف: أقصى وصول
الفئة المستهدفة:
  - العمر: 18-35
  - الجنس: ذكور وإناث
  - الاهتمامات: ألعاب الورق، عملات رقمية، تداول، بلوت
  - المنطقة: مصر، السعودية، الإمارات، الكويت، البحرين، قطر

المحتوى: فيديو 15 ثانية يعرض المنصة

```

```
KPI: CPI < $2
```

```

```
