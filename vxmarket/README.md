# 📁 VEX Marketing Hub — مركز التسويق الشامل

## هيكل الفولدرات

```
vxmarket/
│
├── 📘 README.md                ← أنت هنا
├── 📊 business-plan/           ← خطة العمل وتحليل السوق والمنافسين
├── 🎨 brand-assets/            ← الهوية البصرية والصوتية والهاشتاقات
├── 📅 content-calendar/        ← تقويم المحتوى والمناسبات
│
├── 📱 facebook/                ← كل شيء للفيسبوك
├── 📸 instagram/               ← كل شيء للانستقرام
├── 🎵 tiktok/                  ← كل شيء لتيك توك
├── 🐦 x-twitter/               ← كل شيء لتويتر/X
├── 🎬 youtube/                 ← كل شيء ليوتيوب
├── ✈️ telegram/                ← كل شيء لتليجرام
├── 💬 whatsapp/                ← كل شيء لواتساب
├── 💼 linkedin/                ← كل شيء للينكدإن
├── 👻 snapchat/                ← كل شيء لسناب شات
└── 🎮 discord/                 ← كل شيء لديسكورد
```

## كل فولدر منصة يحتوي على:

| الملف | المحتوى |
|-------|---------|
| `strategy.md` | الاستراتيجية الكاملة للمنصة |
| `posts.md` | منشورات جاهزة للنسخ والنشر |
| `image-prompts.md` | أوصاف لتوليد صور بالذكاء الاصطناعي |
| `video-ideas.md` | أفكار فيديوهات مفصلة |
| `secrets.md` | أسرار الخوارزمية والنمو السريع |
| `ads.md` | استراتيجية الإعلانات المدفوعة (إن وجدت) |

## كيف تستخدم هذا المجلد:

1. **ابدأ بـ** `brand-assets/` — افهم صوت العلامة التجارية
2. **اختر المنصة** — ادخل فولدر المنصة المطلوبة
3. **انسخ المنشورات** — من ملف `posts.md` مباشرة
4. **صمم الصور** — استخدم `image-prompts.md` مع أي أداة AI
5. **تابع التقويم** — من `content-calendar/`
6. **اقرأ الأسرار** — `secrets.md` لكل منصة
