## الشعار (Logo)

```
النسخ المطلوبة:
1. شعار كامل (اسم + أيقونة) — أفقي
2. شعار كامل — عمودي
3. أيقونة فقط (للـ App Icon)
4. نسخة فاتحة (على خلفية غامقة)
5. نسخة غامقة (على خلفية فاتحة)
6. نسخة أحادية اللون (أبيض/أسود)

الأحجام المطلوبة:
- 512x512px (App icon)
- 192x192px (PWA icon)
- 1024x1024px (App Store)
- SVG (للطباعة)
```

## صوت العلامة التجارية (Brand Voice)

### الشخصية:
```
✅ ودّي — نتكلم مع المستخدمين كأصدقاء
✅ شاب — لغة عصرية لكن محترمة
✅ واثق — نعرف قيمة منتجنا
✅ شفاف — نتكلم بصراحة عن كل شي
✅ عربي أصيل — نفتخر بهويتنا العربية
✅ أمين — الأمان والثقة في كل كلمة
```

### النبرة حسب السياق:
```
📢 إعلانات: حماسي + واثق
   "ميزة جديدة مذهلة! 🚀"

📊 تعليمي: واضح + بسيط
   "خطوة بخطوة — كيف تبدأ التداول"

🔒 أمان: جاد + مطمئن
   "أمانك أولويتنا. 89 إصلاح أمني."

🎮 ألعاب: حماسي + مرح
   "كبوت! 😎 أي لعبة تحب أكثر؟"

🔧 دعم: متعاطف + حلّال
   "نفهمك — خلنا نحل المشكلة سوا"
```

### كلمات نستخدمها دائماً:
```
✅ "العب" — "تداول" — "تواصل"
✅ "آمن" — "موثوق" — "مشفّر"
✅ "سهل" — "سريع" — "بسيط"
✅ "عربي" — "مجتمع" — "أصدقاء"
✅ "الأول" — "الوحيد" — "فريد"
```

### كلمات نتجنبها:
```
❌ "أرخص" → "أفضل قيمة"
❌ "مجاني" → "بدون تكلفة"
```
