مفاتيح أساسية (مطلوبة):
المفتاح	الوصف
DATABASE_URL	رابط قاعدة البيانات (موجود تلقائياً)
JWT_USER_SECRET أو SESSION_SECRET	مفتاح تشفير جلسات المستخدمين
JWT_ADMIN_SECRET	مفتاح تشفير جلسات الأدمن
مفاتيح الدفع (Stripe):
المفتاح	الوصف
STRIPE_SECRET_KEY	مفتاح Stripe السري
STRIPE_PUBLISHABLE_KEY	مفتاح Stripe العام
مفاتيح الإشعارات:
المفتاح	الوصف
TWILIO_ACCOUNT_SID	حساب Twilio للرسائل SMS
TWILIO_AUTH_TOKEN	توكن Twilio
TWILIO_PHONE_NUMBER	رقم هاتف Twilio
SENDGRID_API_KEY	مفتاح SendGrid للإيميلات
SENDGRID_FROM_EMAIL	إيميل الإرسال
FIREBASE_PROJECT_ID	مشروع Firebase للإشعارات
FIREBASE_PRIVATE_KEY	مفتاح Firebase السري
TELEGRAM_BOT_TOKEN	توكن بوت تليجرام
مفاتيح تسجيل الدخول الاجتماعي:
المفتاح	الوصف
GOOGLE_CLIENT_ID	معرف Google OAuth
GOOGLE_CLIENT_SECRET	مفتاح Google السري
FACEBOOK_APP_ID	معرف تطبيق Facebook
FACEBOOK_APP_SECRET	مفتاح Facebook السري
TWITTER_API_KEY	مفتاح Twitter API
TWITTER_API_SECRET	مفتاح Twitter السري
مفاتيح إعداد الأدمن:
المفتاح	الوصف
ADMIN_RESET_PASSWORD	كلمة مرور إعادة تعيين الأدمن
ADMIN_RESET_USERNAME	اسم مستخدم الأدمن (افتراضي: admin)
ADMIN_RESET_EMAIL	إيميل الأدمن
