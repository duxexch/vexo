# PWM Design Guidelines - 1xBet Inspired Theme

## Color Palette

### Primary Colors
- **Background**: Dark navy/blue-black (#0f1419 / 210 25% 8%)
- **Card Background**: Slightly lighter (#1a2332 / 215 30% 15%)
- **Sidebar**: Dark (#0d1117 / 215 35% 7%)
- **Primary Accent**: Bright green (#00c853 / 145 100% 39%)
- **Secondary Accent**: Orange/Gold (#ff9800 / 36 100% 50%)
- **Danger**: Red (#f44336 / 4 90% 58%)
- **Warning**: Yellow (#ffc107 / 45 100% 51%)

### Text Colors
- **Primary Text**: White (#ffffff)
- **Secondary Text**: Light gray (#9ca3af)
- **Muted Text**: Gray (#6b7280)
- **Link/Active**: Green (#00c853)

## Typography
- **Font Family**: Inter, system-ui, sans-serif
- **Headings**: Bold, larger sizes
- **Body**: Regular weight, good line height

## Components

### Cards
- Dark background with subtle border
- Rounded corners (rounded-lg)
- Padding: 16-24px
- Hover effect: subtle glow or border highlight

### Buttons
- Primary: Green background (#00c853) with white text
- Secondary: Dark background with border
- Danger: Red background
- All buttons have hover states

### Navigation
- Sidebar with icons and labels
- Active state: green highlight
- Hover: subtle background change

### Balance Display
- Prominent position (top bar)
- Large font size
- Green color for positive
- Quick deposit button nearby

### Game Cards
- Image thumbnail
- Game name overlay
- Play button on hover
- Category badges

### Quick Bet Buttons
- Preset amounts: 10, 25, 50, 100, MAX
- Rounded pills
- Easy to tap on mobile

## Layout Principles
1. Dark theme throughout
2. High contrast for readability
3. Green accents for positive actions (win, deposit)
4. Red accents for negative actions (loss, danger)
5. Mobile-first responsive design
6. Sticky balance bar at top
7. Clear visual hierarchy

## Interactions
- Subtle hover effects
- Smooth transitions
- Immediate feedback on actions
- Loading states for async operations
